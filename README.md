# Dynamic Programming Task Scheduler

A JavaFX desktop application that helps choose the best set of daily tasks within a limited number of available hours.

This project was built for a Design and Analysis of Algorithms course. It uses the Knapsack dynamic programming approach to maximize productivity while staying within the daily time limit.

## Features

- Load tasks from a text file
- Add and update tasks through the GUI
- Set the available number of hours
- Find the optimal task schedule using dynamic programming
- Display the dynamic programming table
- Compare the optimal result with a greedy approach
- Show selected tasks, total time, and total productivity score

## Technologies Used

- Java
- JavaFX
- Maven
- Dynamic Programming
- Knapsack Algorithm
- Greedy Algorithm
- Custom Array List
- File Handling
